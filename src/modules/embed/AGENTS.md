# Embed Module

## Purpose
Public embeddable renderer that can be inserted via iframe. Renders seat maps with click selection, bidirectional postMessage communication with the host page, tooltip display, and theme support.

## Public API
- Components: EmbedRenderer, SeatTooltip
- Hooks: usePostMessage, useSelection

## Routes Owned
- /embed/:id - Public embed renderer

## i18n Key Root
`embed`
