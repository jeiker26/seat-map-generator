# Seatmap Renderer Module

## Purpose
Shared Konva.js-based rendering engine used by both the Editor and Embed modules.

## Public API
- Components: Canvas, SeatComponent, ZoneComponent, Background
- Hooks: useZoomPan, useCoordinates
- Utils: normalizedToPixel, pixelToNormalized, fitToContainer, getStatusColor, generateSeatId

## Routes Owned
None (shared rendering infrastructure)
